HTML (Linguagem de Marcação de Hipertexto)

HTML é uma linguagem de marcação utilizada para estruturar conteudos de páginas na Web, interpretado por navegadores.

CSS (Folhas de estilo em cascata)

CSS é uma linguagem de estilo, usada para personalização visual de elementos escritos em uma linguagem de marcação.

Sass

Sass é um pré-processador que simplifica o desenvolvimento de CSS, reduzindo repetições e aumentando a eficiência do processo.

Bootstrap

Bootstrap é um framework web para componentes de interface responsivos e amigáveis, baseado em HTML, CSS e JavaScript

Taillwind

TaillWind é uma estrutura CSS de código aberto , permite que você escreva CSS em seu HTML na forma de classes pré-definidas.

Javascript

Javascript é uma linguagem de programação interpretada estruturada , baseada em objetoss, tornando o layoout web mais dinamico.

VUE

VUE é um Framework JavaScript para aplicações web rápidas, com renderização declarativa e composição de componentes
